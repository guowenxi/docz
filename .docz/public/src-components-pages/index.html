项目基础配置

单个菜单路由跳转后的主页面配置,每一份配置路径在src/pages/#{主页面名}/columns.json

url

列表请求的地址

params

列表请求的参数

checkBox

列表是否带有多选功能

operation

列表内操作按钮的配置

(item)

name按钮名称
type按钮类型
drawerWidth按钮弹出宽度

searchListType

列表查询样式类型

  • 可选择参数 : complexQuery | singleQuery | multipleQuery
singleQuery简单的只带关键字的查询
complexQuery复杂的带关键字的查询
multipleQuery单行多条查询

searchList

列表查询配置数组

(item)

title查询名称
type按钮类型
key字段名
placeholder默认文字
type 类型
  • 可选择参数 : input | select | period | number-interval | input-number | time-period | month-period | select-Level
input输入框
select选择框
period时间范围
number-interval数字范围
input-number数字输入框
time-period日期时间段范围
month-period月份时间段范围
select-Level选择等级

tableColumns

设置列表字段的配置

(item)

title字段映射中文
type按钮类型
key字段名
dataIndex字段名
width宽度(带百分比)
align对齐方式
