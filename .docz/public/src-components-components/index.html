详情组件

一些常用组件,的配置说明, 主页面的详情组件在src/pages/#{主页面名}/TABLEJSON.json内

*TABLEJSON.json是以数组的形式保存,如果有多个详情按钮,这里的数组将会是多维

详情组件通用字段

name关联的字段名
width组件在详情里的宽度
label组件标题配置
name组件标题名
*col标题所占份数(总为10份)默认值 3
*style标题样式默认值 {}
field组件内容配置
*col内容所占份数(总为10份)默认值 7
type内容类型(见其他类型配置)默认值 NULL
props内容类型相关配置默认值 NULL
*defaultValue内容类型相关配置默认值 ''
*placeholder默认值默认值 NULL
*relateNames多组件联动的name默认值 {}
*style内容自定样式默认值 {}
rules是否必填配置默认值 NULL
required是否必填(boolean)默认值 true
message必填的提示文字默认值 ''

*relatekeys需要联动组件的字段数组(字符串)默认值 ''
